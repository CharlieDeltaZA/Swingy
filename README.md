### Installing & Running:
Ensure you have the latest JRE & JDK (14 at time of development), as well as Maven.

Build with Maven:
```
mvn clean package
```

Run with Java, specificying which interface you would like to use:
```
java -jar ./target/swingy.jar [console/gui]
```
